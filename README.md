<The South>는 <The North's panda>와 같은 플레이 방식을 공유하는 게임입니다.

하지만 더욱 좋은 경험을 위해 UI와 기능을 대폭 수정했습니다.

 세이브 기능
이제부터는 자동적으로 게임 진행 상태가 세이브 되어, 게임 로드를 통해 이어 할 수 있습니다.
 개체 선택 방식 수정
이전까지 모든 개체를 클릭 선택해야 하기에 불편했던 점을 개선하여 드래그 선택 방식으로 수정하였습니다.
 화면 레이아웃
이전까지 모바일 또는 스크린 크기가 다른 기기에서 플레이할 때 개체의 속도가 빨라지는 일,  또는 UI 배치가 꼬이거나 과도하게 작아져 불편해지는 일이 있었습니다.
이번 <The South>에서는 이를 수정하여 자동으로 사용자의 레이아웃에 맞게 UI배치와 개체 속도를 조절하는 기능을 도입하였습니다.


많은 이용 부탁드립니다.
